# Demello Design Co. — Portfólio

Página de portfólio temporária enquanto os cases são desenvolvidos.  
Tema padrão: **dark mode**. Visitantes podem alternar pelo botão no header.

| Arquivo | Descrição |
|---|---|
| `index.html` | Página principal |

| `ddc-horizontal-black.svg` | Logo para o tema claro |

---

## Como publicar no GitHub Pages

### 1. Crie o repositório

- Acesse [github.com](https://github.com) e faça login
- Clique em **"New repository"**
- Em **Repository name**, digite exatamente: `seuusuario.github.io`
  _(substitua pelo seu username do GitHub — ex: `juliodemello.github.io`)_
- Marque como **Public**
- Clique em **"Create repository"**

### 2. Suba os arquivos

- Clique em **"Add file" → "Upload files"**
- Arraste os **4 arquivos** juntos:
  - `index.html`
  - `avatar2026-800x800px.png`
  - `ddc-horizontal-white.svg`
  - `ddc-horizontal-black.svg`
- Clique em **"Commit changes"**

### 3. Ative o GitHub Pages

- Vá em **Settings → Pages**
- Em **Branch**, selecione `main` e clique em **Save**

### 4. Acesse o site

Aguarde 1–2 minutos e acesse:

```
https://seuusuario.github.io
```

---

## Toggle de tema

O botão lua/sol no header alterna entre dark e light mode.  
A preferência fica salva no navegador do visitante via `localStorage`.

| Tema | Logo carregada |
|---|---|
| Dark (padrão) | `ddc-horizontal-white.svg` |
| Light | `ddc-horizontal-black.svg` |

---

## Atualizando o site

- No repositório, clique no arquivo a substituir
- Clique em **"..."** → **"Replace file"**
- Confirme o commit — alterações ficam no ar em menos de 1 minuto

---

## Domínio personalizado (opcional)

1. Compre um domínio (Registro.br, GoDaddy, Namecheap, etc.)
2. Em **Settings → Pages**, adicione em **"Custom domain"**
3. No registrador, crie um `CNAME` apontando para `seuusuario.github.io`
